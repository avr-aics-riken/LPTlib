LPTlib

Lagrangian Particle Tracking library is a C++ class library to compute trajectories of numerous particles. This library is designed to achieve higher scalability in large-scale parallel environments using particle-based load balancing and data caching algorithm to hide the communication cost of data transfer.

Software Requirement

  • MPI library
  • Text parser library
